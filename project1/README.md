# 时间线网站 (Timeline Website)

一个美观、交互式的时间线网站，允许用户添加、查看和管理时间点事件。采用天蓝色海洋主题设计，具有流畅的展开/收起动画效果。

## 📋 项目简介

这是一个纯前端的时间线管理网站，用户可以通过简单的操作添加时间点，每个时间点可以包含日期、标题、描述和图片。点击时间点可以展开查看详细信息，再次点击则收起。所有数据保存在浏览器的本地存储中，无需后端服务器。

## ✨ 功能特性

### 核心功能
- ✅ **添加时间点**：通过表单添加新的事件时间点
- ✅ **展开/收起**：点击时间点展开显示详细内容（照片和描述）
- ✅ **自动排序**：时间点按日期自动排序显示
- ✅ **删除功能**：每个时间点都可以单独删除
- ✅ **数据持久化**：使用 localStorage 保存数据，刷新页面不丢失

### 界面特性
- 🎨 **天蓝色海洋主题**：美观的渐变背景，模拟海洋效果
- 🔤 **硬体字（等宽字体）**：使用 Courier New 等宽字体，整体风格统一
- 💫 **流畅动画**：展开/收起带有平滑过渡效果
- 📱 **响应式设计**：支持桌面和移动设备
- 🌊 **毛玻璃效果**：卡片使用半透明背景和模糊效果

## 🛠️ 技术栈

  - Flexbox 布局

## 📖 使用说明

### 添加时间点

1. 点击页面右上角的 **"+ 添加时间点"** 按钮
2. 在弹出的表单中填写：
   - **日期**：选择或输入日期（必填）
   - **标题**：输入事件标题（必填）
   - **描述**：输入详细描述（可选）
   - **图片URL**：输入图片的网络链接（可选）
3. 点击 **"添加"** 按钮完成添加

### 查看时间点

- **展开**：点击任意时间点，会展开显示该时间点的详细内容（描述和图片）
- **收起**：再次点击已展开的时间点，会收起详细内容
- **自动关闭**：展开一个时间点时，其他已展开的时间点会自动收起

### 删除时间点

1. 点击要删除的时间点，展开其内容
2. 点击内容区域底部的 **"删除"** 按钮
3. 在确认对话框中点击 **"确定"** 完成删除

### 数据管理

- 所有数据自动保存在浏览器的 **localStorage** 中
- 数据会一直保留，直到：
  - 清除浏览器缓存
  - 手动删除浏览器数据
  - 使用浏览器的开发者工具清除 localStorage

## 🔧 代码说明

### 主要函数

#### `addTimelineItem()`
添加新的时间点到时间线，自动按日期排序。

#### `deleteTimelineItem(id)`
删除指定ID的时间点，需要用户确认。

#### `toggleTimelineItem(id)`
切换时间点的展开/收起状态，同时关闭其他已展开的时间点。

#### `renderTimeline()`
渲染整个时间线，根据数据动态生成HTML。

#### `saveTimelineData()`
将时间线数据保存到 localStorage。

#### `formatDate(dateString)`
格式化日期字符串为 YYYY-MM-DD 格式。

#### `escapeHtml(text)`
转义HTML特殊字符，防止XSS攻击。

### 数据结构

每个时间点对象包含以下字段：
```javascript
{
    id: Number,        // 唯一标识符（时间戳）
    date: String,      // 日期（YYYY-MM-DD格式）
    title: String,     // 标题
    description: String, // 描述（可选）
    image: String      // 图片URL（可选）
}
```

## 📝 注意事项

1. **图片URL**：必须使用有效的网络图片链接（http:// 或 https://）
   - 本地图片文件需要使用文件服务器
   - 可以使用图床服务（如 imgur、sm.ms 等）

2. **数据备份**：重要数据建议定期导出备份
   - 可以通过浏览器开发者工具 → Application → Local Storage 查看数据
   - 可以手动复制 JSON 数据进行备份

3. **隐私**：数据仅存储在本地浏览器，不会上传到服务器
